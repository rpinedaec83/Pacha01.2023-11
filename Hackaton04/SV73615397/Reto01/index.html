¿Cómo defines una función?

Podemos definirlo declarando el bloque function(params){//codigo} o usando la expresión const myFunction = (params)=>{//codigo}

¿Hasta cuantos argumentos puedes declarar en una función?

No hay un limite como tal, pero es recomendable no hacer más de las necesarias, ya que una cantidad excesiva de argumentos dificultaría la lectura de nuestro código.
